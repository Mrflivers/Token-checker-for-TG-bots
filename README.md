# 🔐 Telegram Token Checker

Веб-приложение для проверки валидности Telegram Bot токенов с удобным интерфейсом.

## ✨ Функциональность

- **Проверка токенов**: Автоматическая проверка Telegram Bot токенов через официальное API
- **Загрузка файлов**: Поддержка загрузки `.txt` файлов с токенами
- **Извлечение токенов**: Автоматическое извлечение токенов из любого текста по регулярному выражению
- **Детальная статистика**: Отображение информации о валидных и невалидных токенах
- **Информация о ботах**: Для валидных токенов показывает username, имя и ID бота
- **Система авторизации**: Защищенный доступ с логином и паролем
- **Современный интерфейс**: Адаптивный дизайн с темной темой

## 📋 Требования

- Python 3.8+
- pip

## 🚀 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Mrflivers/Token-checker-for-TG-bots
cd telegram-token-checker
```

2. Создайте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/Mac
# или
.venv\Scripts\activate  # Для Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Настройте учетные данные администратора в `app.py`:
```python
ADMIN_USERNAME = 'your_username'
ADMIN_PASSWORD = 'your_password'
```

## 🎯 Использование

1. Запустите приложение:
```bash
python app.py
```

2. Откройте браузер и перейдите по адресу:
```
http://localhost:8080
```

3. Войдите в систему, используя настроенные учетные данные

4. Загрузите файл с токенами (`.txt` формат) или вставьте токены вручную

5. Просмотрите результаты проверки

## 📝 Формат токенов

Приложение автоматически извлекает токены из текста по шаблону:
```
123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

Где:
- `123456789` - ID бота (8-10 цифр)
- `:` - разделитель
- `ABCdefGHI...` - токен (35 символов: буквы, цифры, подчеркивания, дефисы)

- Загруженные файлы автоматически удаляются после проверки

## 🐛 Возможные статусы токенов

- **active** - Токен валиден и бот активен
- **invalid** - Неверный формат токена
- **unauthorized** - Бот заблокирован или не запущен
- **forbidden** - Бот был заблокирован
- **error** - Ошибка при проверке
